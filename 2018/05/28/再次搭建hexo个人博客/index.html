然而这个博客还是存在很大的不足，比如每次打开总是要从命令行输入
hexo -s，再通过localhost：4000进入，这样每次登陆自己博客显得太
低端了。而且在日常使用中，只能在本机登陆对于我们的使用也是显得
很不方便。同时这种情况下其他人没有办法访问，也失去了其分享交流
的意义。
